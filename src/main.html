Drag a file or Click Open

Press view to preview the markdown.

Press edit to edit the source text.

Suported markdown elements:

  • # to make a new title.
  • ## or ### to sub-titles.
  • - to list items

    • Note that only putting a space next to this modifiers really apply them.

    Suported markdown inline formatting:

  • ** to make inline bold: **BOLD TEXT**
